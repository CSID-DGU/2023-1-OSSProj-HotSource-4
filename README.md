# 프로젝트 내용 : 
- 이클래스와 유사한 페이지를 구현하고 팀활동 페이지에 메모 기능과 채팅 기능을 포함한 협업 기능을 추가 
- 관리자 계정(교수님 계정)으로 접속하여 수강생들을 선택하여 팀 생성 가능
- 학생 계정으로 접속하여 본인에게 부여된 팀을 조회하고 팀활동 modal 창에서 협업 기능 사용 가능
